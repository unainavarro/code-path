## Listas de Definiciones

Para mostrar listas de definiciones en HTML se utiliza las etiquetas: <br>
`<dl> <dt>Término<dt> <dd>Definición<dd> </dl>`

```HTML
Ejemplo:
<dl>
  <dt>Término1</dt>
  <dd>Definición 1</dd>

  <dt>Término 2</dt>
  <dd>Definición 2</dd>
</dl>

Resultado:
    Término1
        Definición 1
    Término 2
        Definición 2
```
